# Glossary questions (ask only whatâ€™s missing)

Use these when naming is drifting or when readers could be confused.

## Minimum set (usually enough)

1) **What is the term?**
- Definition in one sentence.

2) **Where is it used?**
- Which contexts should use this term consistently?

3) **What is it not?**
- Any near-synonyms or overloaded meanings to disambiguate?

4) **What must we call it?**
- Preferred name/spelling and any required alias.

5) **What must we avoid calling it?**
- Forbidden names that cause confusion or drift.
